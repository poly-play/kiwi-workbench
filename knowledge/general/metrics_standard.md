# 关键指标定义标准 (Global Metrics Standard)

为了确保跨部门沟通的一致性，定义以下核心指标的计算口径。

## 活跃度指标
### Active User (活跃用户)
*   **定义**: 当日有**成功投注 (Bet)** 行为的用户。
*   *排除*: 仅登录未游戏、仅充值未游戏。
*   **DAU**: Daily Active Users (日活跃用户数)
*   **MAU**: Monthly Active Users (月活跃用户数)

## 交易指标
### Deposit Success Rate (充值成功率)
*   **定义**: `成功充值单数 / 发其充值请求总单数`。
*   *注意*: 排除用户未进入支付页面的点击行为，仅计算进入三方网关的请求。

## 红利指标
### Bonus Cost Ratio (红利成本率)
*   **定义**: `发放红利总额 / GGR`。
*   *健康值*: 通常控制在 20% - 30% 以下，视具体业务阶段而定。
